# 募集文ジェネレーター

クラウドワークス・ランサーズ向けの募集文をAIで自動生成するツールです。

## 機能

- クラウドワークス / ランサーズ対応
- カテゴリ・サブカテゴリ選択
- 募集トーン選択（作業者募集 / スタンダード / パートナー募集）
- 音声入力対応（プッシュトゥトーク）
- 生成文のワンクリックコピー

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env.local` ファイルをルートに作成して以下を記載：

```
ANTHROPIC_API_KEY=your_api_key_here
```

APIキーは [Anthropic Console](https://console.anthropic.com/) で取得できます。

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方

1. プラットフォームを選択（クラウドワークス or ランサーズ）
2. カテゴリを選択
3. 支払い方式・募集期日を選択
4. 作業内容・必要スキルを入力（音声入力も可）
5. 「募集文を生成する」をクリック
6. 生成された文章をコピーして貼り付け
